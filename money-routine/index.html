자주 묻는 질문

데이터를 백업하거나 복원하려면 어떻게 하나요?

앱의 설정 → 데이터 백업 / 복원에서 처리할 수 있습니다. 내보낸 백업 파일을 클라우드 드라이브에 저장해 두면 기기를 바꿔도 그대로 이어 쓸 수 있습니다.

결제일 알림이 오지 않습니다.

기기 설정 → 알림 → 머니루틴에서 알림 권한이 켜져 있는지 먼저 확인해 주세요. 앱의 설정 → 납부일 알림에서 알림을 활성화하고 알림 시각을 다시 저장하면 곧바로 다음 결제일부터 발송됩니다.

새 기기로 데이터를 옮기는 방법은?

이전 기기에서 설정 → 데이터 백업 / 복원 → 백업 파일 내보내기로 파일을 만들고, 새 기기에서 머니루틴을 설치한 뒤 같은 화면에서 가져오기로 그 파일을 선택하시면 됩니다.

고정지출을 잘못 등록했어요.

고정지출 템플릿 화면에서 해당 항목을 탭하면 수정·삭제할 수 있습니다. 월별 화면에서 보이는 항목은 템플릿의 사본이므로, 템플릿을 수정하면 다음 달부터 자동 반영됩니다.

여러 결제를 한 번의 자동이체로 묶고 싶습니다.

월별 고정지출 화면에서 항목을 길게 눌러 선택한 뒤, 상단의 그룹 묶기 메뉴를 사용하시면 됩니다. 실제 카드 명세서 단위와 동일하게 보이도록 정리할 수 있습니다.

앱이 갑자기 종료되거나 동작이 이상합니다.

앱을 완전히 종료한 뒤 다시 실행해 보시고, 같은 증상이 반복되면 아래 이메일로 사용 중인 기기·OS 버전과 발생 시점을 알려주세요. 크래시 보고서가 있다면 함께 보내주시면 더 빠르게 확인할 수 있습니다.

내 데이터는 어디에 저장되나요?

모든 데이터는 사용자의 기기 내부 데이터베이스(SQLite)에만 저장됩니다. 별도의 서버 업로드나 회원가입이 필요하지 않습니다. 자세한 내용은 개인정보처리방침을 참고해 주세요.
